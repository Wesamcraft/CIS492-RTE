Welcome to Our Class Project

This website showcases our class project for the Real-Time Embedded Systems course at Cleveland State University.

Project Overview

Project Goal: We were tasked with the construction and operation of the SonicModell AR Wing Pro 1000mm. The glider had to fly a specified distance to take photos of a target object. The photos were then sent to another team so they could use their own rover to retrieve the target.
